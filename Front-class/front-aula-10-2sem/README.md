
# Título do projeto
---
```
Miniportal de Clientes
```

## Tecnologias utilizadas.
1) Vite
2) React
3) Typescript
4) nodeJS
5) CSS

## Descrição das funcionalidades.
1) GetAll  - Listar todos os clientes em cards, possibilitando a escolha de um para realização de análise.
2) GetById - Selecionar um único cliente e apresentar seus detalhes em um único Card, possibilitando realizar uma análise mais criteriosa.

## Passos para rodar o projeto.
1) Utilizar um terminal de comando.
2) Windows: CMD
3) Linux e MAC: Bash
4) Rodar o seguinte comando: npm run dev
5) Acessar a url: http://localhost:5173
