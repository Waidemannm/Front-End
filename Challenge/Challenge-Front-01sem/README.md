# Challenge-Front

## 📚 Estrutura do Projeto
# 🩺 Assistente de Saúde Digital Acessível – HC Lembrete Fácil

## 📌 Objetivo do Projeto

Desenvolver uma solução digital simples, inclusiva e eficaz para **reduzir a abstenção de pacientes nas consultas e exames do Hospital das Clínicas**, com foco em **idosos e pessoas com dificuldades no uso de tecnologia**.

---

## 💡 Problemas Identificados

- **Alta taxa de abstenção (faltas):**  
  Pacientes esquecem datas de consultas e exames, especialmente idosos que não utilizam aplicativos. Isso causa buracos na agenda médica e aumento das filas.

- **Dificuldade com tecnologia (apps/senhas):**  
  Muitos pacientes têm dificuldades com logins, senhas ou uso de apps de saúde. Isso gera barreiras de acesso a serviços essenciais como resultados de exames ou consultas online.

- **Baixa adesão à telemedicina:**  
  Consultas online são perdidas porque pacientes não sabem como acessá-las, ou não conseguem entrar no horário marcado.

---

## 🚀 Solução Proposta

Criar um sistema acessível que combina:

- ✅ **Assistente Virtual via WhatsApp e Web (chatbot)**  
  Envia lembretes automáticos com linguagem simples, permitindo confirmação ou cancelamento com um clique/resposta.

- ✅ **Página Web Acessível**  
  Interface simples onde o paciente pode visualizar suas consultas e acessar links de telemedicina com tutoriais básicos.

- ✅ **Automação com Python/Java**  
  Envio de mensagens, recebimento de respostas e atualização de presença no banco de dados.

- ✅ **Banco de Dados Centralizado**  
  Para armazenar pacientes, agendamentos, confirmações e logs de interações.

## 🔍 Funcionalidades Principais

- 📅 Envio de lembretes automáticos de consultas e exames.
- ✅ Confirmação ou cancelamento com simples resposta (ex: "1" para confirmar).
- 🧓 Interface pensada para idosos e pessoas com baixa letramento digital.
- 🔗 Link direto para telemedicina + tutorial com imagens simples.
